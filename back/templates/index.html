Добро пожаловать в Quiz App API

Этот API позволяет вам взаимодействовать с приложением викторины.

Запрос: GET /api/questions

Описание: Получает список вопросов викторины.

Пример использования:

fetch('/api/questions').then(res => res.json()).then(data => console.log(data));

Запрос: POST /api/submit-answers

Описание: Отправляет ответы пользователя на вопросы и возвращает результат.

Пример использования:


fetch('/api/submit-answers', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        user_id: 1,
        answers: [
            {"question_id": 1, "answer": "Python"},
            {"question_id": 2, "answer": "Java"}
        ]
    })
}).then(res => res.json()).then(data => console.log(data));
